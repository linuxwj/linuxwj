专访HTH创始人：项目制学习好在哪？ - FT中文网

2017-10-22
项目制学习

更新于2017年10月18日 06:14 丁梦琪 黄剑桥 为FT中文网撰稿
19-23 分钟

【编者按】“教育创新”专题由FT中文网与“中美对话”项目组联手推出。通过专注于对教育理念、方法及科技三个层面的深度挖掘，展示或激动人心，或发人深思的案例，更期待能抛砖引玉，以回应快速变化的时代给未来人才培养提出的更具挑战性的要求。
没有教科书，没有上下课铃声，所有的学生都是从学区中抽签随机录取。如此颠覆传统的学校，却拥有98%的本科升学率和颇高的名校录取人数。这样的教育奇观，引得无数人前来High Tech High这所位于美国的项目制学习鼻祖学校取经。作为High Tech High的共同创始人，Rob Riordan全程参与了High Tech High教育理念的确定、项目制学习的设计，及教师培训系统的建成。他言谈冷静又智慧，在与中美对话的交谈之间，充盈着对项目制学习的激情。
一
“我们2000年九月创立High Tech High时，我们知道我们想要这样一所学校：当你走进它，你说不出它是一所技术学校还是一所大学预备学校。当你在廊道里行走，你可以看到学生们戴着护目镜，使用着电动工具。但这并不是技工课，而是九年级的数学课、物理课、艺术课甚至人文课。”Rob Riordan向中美对话介绍道，“这种项目制学习的方法，给学生提供了很多闪光的方式和不同的入门点，并且从更大程度上来说，能够促进教育公平：当来自不同背景、拥有不同学术经验的学生并肩坐在教室，携手完成项目，每个人都能受益。”
于是，High Tech High作为一所公立特许高中在圣地亚哥城被建立了起来，Rob也把他梦想中的“无界限”项目制学习落实到了学校的教学实践中：
•学科间无界限，学校没有明确的学科，只有深度的项目研究，学生通过项目展开个性化学习（personalized learning）；
Facebook个性化学习平台，有多个性？
黄剑桥：Facebook技术支持下的公立学校Summit将96%的学生送进了大学。其个性化学习平台是如何运转的呢？

•课程无界限，没有大学先修课程（Advanced Placement course，简称AP课），也没有分开的荣誉课程（honors classes），而是所有高低年级的核心课都提供课内的荣誉选项；
•学生生活与成年生涯无界限，学习与工作结合，学生们在实习、实践中为未来做准备；
•不同条件的学生无界限，用抽签方式录取多元化、不同水平的学生，直接对应圣地亚哥的学龄人口组成；
•学生在学习中无界限，所有项目通过小组紧密合作的形式完成；
•老师与学生无界限，学生不仅仅是项目的执行者，师生双方是项目的共同设计者（co-designers），在项目中共同探讨、共同成长；
•年轻老师与老一辈的老师也无界限，老师们在相互的学习、讨论中共同成长。
Rob说：“这所面向未来的学校无视了20世纪学校教育中的很多原理，比如把学生分为不同的学术路径、把知识分为不同的科目、把学校与真实世界分离。相反，我主张把学生整合起来，让他们浸入跨学科学习，并与社区紧密联系。”
如今，随着High Tech High模式的成功，High Tech High已经在圣地亚哥开办了13所学校，其中包括四所小学，四所初中，和五所高中。
二

High Tech High的学生正在制作项目
Rob Riordan对中美对话说，在High Tech High，课堂以问题开始，自己更重视问题而不是答案。在High Tech High老师Bobby Shaddox 和Allie Wong共享的课堂上，出现了这样的一幕：
“对于世界、对于你的生命，大家有怎样的问题、忧虑或者好奇？”老师们先问。坐在讲台下的45名学生来自不同的族裔，有着不同的社会经济和学术成就情况，他们不约而同地盯着白板，然后十分兴奋地在笔记上写下了自己的问题。在随后的25分钟里，孩子们思考、求索，然后开始与小组内的同学们分享自己的问题：
“我们是怎样把盐水变成饮用水的？”
“我们怎样才能制造一辆可以在垃圾上跑的车？”
“我们怎样才能阻止全球变暖？”
“为什么我很生气？”
“为什么人们会互相伤害”
……
随着学生提出一个个问题，几个清晰的主题在课堂中浮现了出来，例如，生命的起源、世界的终结、及人类的角色。在数日的讨论过后，全班同学们提取出了一个最本质的问题，作为他们共同项目的选题：世界可能以怎样的方式终结？以及人类可以做什么来进行预防？
在随后的三个月里，学生们结成小组，在这一大问题下进行探索。这些问题联系起了科学、数学、人文等多种学科，这些也是Bobby老师和Allie老师所负责的“学科丛”。不过，这些问题并不是某个学科的入门问题，而是生命问题–有的学生探索起了黑洞，有的研究起了玛雅日历、台风、火山、战争、气候变化、流星影响等等。为了获取资料和数据，学生们去寻找现存的资源，通过在线或面对面的方式与印度、夏威夷的专家交谈，或是去到大学、博物馆、社区中学习。
最后，学生们用自己写的文字和绘制的艺术组成了一本书，经过几轮相互修订和检查之后，出版了《揭秘世界终结（The End of the World Uncovered）》一书，并组织了一次公共展览，用于介绍他们的发现和书籍。因为这次展览，很多学生持续地改进自己的项目作品，许许多多的家长和朋友们也参与了进来。
Rob Riordan对中美对话说：“这些项目触发了深度学习。学生们可能没有那么多的课程选择，但是当他们通过做项目来探索问题的时候，他们也有丰富的选择。我希望，当人们问学生他们在学什么的时候，他们能够深入地描述他们的项目，而不是说我早上9:04上了数学课，9:52上了英语课。基于项目制学习的简单、整合式结构，教师们也能更深入地了解学生，并在教室里建立起一个学习社区”。
三

High Tech High的师生共同协作项目
在Rob Riordan和他的项目制学习里，学生们在跑项目时，老师们也没闲着。从入校的那一刻起，Rob就为老师们安排了一次全新的成长旅程。Rob说：“不论是刚刚拿到教师资格证的毕业生、转行从事教育的人，还是有多年教书经验的老教师，都会在开学前参加一个七天的‘奥德赛’项目，以获得High Tech High模式的设计原则、教学方法的浸入式体验。”
在“奥德赛”的第一天，教师们就会展开一项为期两天的“小项目”，在其中，老师们重新变成学习者，充分体验着跨学科学习项目如何运作。在每个项目中，教师们所做的活动，都为其此后引导学生进行项目制学习进行了准备：老师们需要提出问题，在社区中进行田野实践，并且在项目修订和改正之后组织一次展览。随后，教师们被聚集在一起，分享自己认为的组成“有意义的学习”的关键要素，并畅想如果在一个地方“有意义的学习”一直在进行，这个地方应该长成何种模样。
在老师开始设计他们的第一个项目前，按照Rob的设计，老师们需要参加以“项目调试”和“检视学生作品”为主题的培训。一名老教师会带着一份项目设计的草稿或是学生作品的范例，前来向新老师寻求建议。新老师、旧老师会形成小组，新老师依照采访提纲提出事实性和探索性的问题，并就作品展开一系列对话。
Rob说：“这些培训能让老师们感受到自己观点的重要性，并且正视到作为老师，应当把自己的作品和想法公开，最重要的是，自己仍然是学习者。此外，新老师也能够学习到对项目的评判标准，据此来指导自己今后的项目设计。”最终，老师们以一场正式的学习展示来结束自己的旅程，他们向其他新旧老师分享自己的项目设计、探讨现存的问题，这也是今后他们指导学生所必须经历的环节。
在七天的“奥德赛”之后，Rob向中美对话说，教师们的努力也从未停止。他们用八天来跟随老教师进行实地学习，也每天都比学生早到一个小时，不为别的，只为相互探讨教学。在Rob担任荣誉主席的High Tech High 教育研究生院里，老师们攻读教师证书项目，学习方法论、项目设计、教学平等和多元化等等。
Rob说：“成人的学习反射出学生的学习，这也是High Tech High模式的一个重要特点”。“奥德赛（Odyssey）”意为“漫长而刺激的奇幻历程”，从Rob的设计看来，这个名字不足为怪。
四

Rob Riordan在High Tech High演讲
在High Tech High的教育影响力之路上，仍存在许多束缚和一些质疑的声音。《纽约时报》专栏作家David Brooks曾发表《为智慧而生的学校（Schools for Wisdom）》一文，指出High Tech High模式可能跳跃了学生在构建知识和智慧时所必经的过程——事实的获取及文化素养的培育。在David看来，知识的获取必须有基本的事实积累，例如，学生要知道什么是基因，什么是中子。其次，模式需要形成。在课堂上，好的老师需要来引导课堂讨论，帮助学生把知识联系起来形成模式。而在High Tech High,学生们扎根于项目，可能忽略了基本知识的学习和传统课堂的意义。
Rob则认为High Tech High模式与传统教育的区别只是在于教育的起点：“在High Tech High，我们鼓励老师从对一个项目的想法开始，对于这个项目，他们自己要有激情，这个项目也要能让学生投入进去。在这个起点之后，我们会融入传统的内容和课程标准。我知道学校需要遵照加州的课程标准，所以我们从有趣的问题开始，设计项目，然后参照课标来看我们做到了哪些。对于遗漏的重要课程标准，我们会让老师在另一个项目里强调，或者通过别的方式来教授，比如直接的讲课。另外，我们的老师是项目设计师，学生是共同设计师，很多时候由老师来完成最终的项目设计，然后把项目交给学生，所以很多项目是规定好的，学生在执行项目的过程中只是从课程外引入了更多的资源。”
另外，在美国学校类点评网站Greatschools.com上，出现了不少对High Tech High系列学校的负面评价。一位家长的小孩曾就读于High Tech系列的媒介艺术学校，她说：”当我发现我女儿在加州州立大学第一年就需要补数学、科学和英语的时候，我震惊了。她当年以高于3.5的GPA从High Tech媒介艺术学校毕业。我联系她在加州州立大学的学术顾问时，那位老师说‘从High Tech媒介艺术学校来的学生只有1%为大学做好了准备。“
Rob Riordan则反驳道，High Tech High 为学生升入更高一级的学校，甚至终身学习做好了充足的准备。首先，在大学入学方面，High Tech High学生的平均分数高于全市、全街区或者全州的平均分，分散到每个族裔、每种收入情况的学生上来看也仍然很高。Rob向中美对话介绍，学校会分配少部分的时间帮助学生准备SAT和ACT，不过“学校依然力图弱化考试，因为考试只代表治理水平很窄、很浅的一部分”。其次，在学习习惯上，High Tech High在项目中教授了学生进行选择、把控自己、掌握学习系统的能力。Rob说，“从High Tech High毕业出去的学生，在面对学术困难时，敢于向教授询问，会在教授的办公时间与教授探讨，会组建学习小组相互帮助，而且明确地知道自己为什么会来读大学，知道自己的目的。有这样能力和习惯的学生更容易成功。”最后，在终身学习方面，Rob认为，High Tech High为学生提供了充分的机会来进行选择、尝试，他们能够拥有毕业后没有的机会，来体验不同的选择。这样一来，带着他们学到的技能和他们形成的性情、选择，他们有能力来应对不确定的未来。
不过，除了这些，Rob Riordan仍然有很多忧虑，这些忧虑也成为了High Tech High模式进一步拓展的绊脚石。例如，师资方面，很多老师还是有传统的对自己科目的“权威感”，认为自己是课堂的权威，并且自认很了解自己的科目，这关系到老师对自己身份的认同感。于是，跨学科、共设计的项目制学习无法展开，深度学习的效果也无法达到。另外，目前世界各地很多课堂的规模非常大，每个学生也有不同的选课，这就导致教师很难做项目。Rob对中美对话说：“如果有更少的学生，更多的时间做项目就很容易。在High Tech High，不同老师共享几个跨学科科目，节省了时间；一小组学生共享一个项目，由同一个老师领导，相对减少了学生的数量。”最后，如何既保持项目制学习的传统，又向创新的观点全面开放，是Rob认为High Tech High今后发展中非常需要平衡的问题。
